# BadUSB on Arduino

## What do i need to prepare??
請先自備：<br>
**Arduino Leonardo Micro**<br>
**MicroUSB to USB**轉接頭或傳輸線<br>
**FTP站台**<br>
**Windows7**電腦<br>
**Arduino IDE**<br>
